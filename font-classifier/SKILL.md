# Font Classifier

## Features
- Classify 30+ common web fonts by typographic category
- Suggest pairings based on contrast principles
- Categories: geometric-sans, humanist-sans, neo-grotesque, transitional-serif, old-style-serif, slab-serif, didone-serif, monospace

## Quick Start
```bash
node scripts/classify.js "Inter"
node scripts/classify.js "Playfair Display"
```

## Use when
Choosing font pairings, classifying existing font stacks, building font pairing tools.
