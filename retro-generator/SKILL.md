# Retro Generator

## Features

- 5 retrospective formats with facilitator guides
- Time-allocated agendas
- Prompting questions per section
- Action item template

## Quick Start

```bash
node scripts/generate.js --format start-stop-continue
node scripts/generate.js --format sailboat --duration 60
node scripts/generate.js --list
```

## Use when

Sprint retrospectives, project post-mortems, team reflection sessions.
