# Task Prioritizer

## Features

- Eisenhower matrix (urgent x important)
- Keyword-based scoring
- Quadrant assignment: Do First, Schedule, Delegate, Eliminate
- Sorted output with priority labels

## Quick Start

```bash
node scripts/prioritize.js "Fix login bug; Write blog post; Update docs; Plan Q3 roadmap"
node scripts/prioritize.js tasks.txt
```

## Use when

Overwhelmed with tasks, sprint planning, deciding what to work on next.
