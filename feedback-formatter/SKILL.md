# Feedback Formatter

## Features

- SBI framework with prompts
- Tone types: constructive, appreciative, corrective
- Complete template with support/suggestion sections

## Quick Start

```bash
node scripts/format.js "Sarah interrupted during standup" --type corrective
node scripts/format.js "Alex shipped ahead of schedule" --type appreciative
```

## Use when

Performance reviews, 1:1 prep, giving difficult feedback, documenting observations.
