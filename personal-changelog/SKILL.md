# Personal Changelog

## Features

- Auto-pull from git commits
- Categories: Shipped, Learned, Connected, Started
- Weekly and monthly summaries
- Markdown output for blog or journal

## Quick Start

```bash
node scripts/generate.js --git --period week
node scripts/generate.js --manual "Shipped auth; Learned GraphQL; Met 3 new people"
```

## Use when

Friday reflection, portfolio updates, performance review prep, tracking growth.
