# Journal Prompt

## Features

- Time-aware (morning reflection, evening review)
- Focus areas: gratitude, goals, creativity, relationships, growth
- Random + curated selection
- 40+ unique prompts

## Quick Start

```bash
node scripts/generate.js
node scripts/generate.js --focus gratitude
node scripts/generate.js --evening --count 5
```

## Use when

Starting a journaling habit, breaking through writer's block, structured self-reflection.
