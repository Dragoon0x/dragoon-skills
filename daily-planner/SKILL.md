# Daily Planner

## Features

- Morning planning with intention setting
- Time-blocked schedule by energy level
- Top 3 priorities
- Evening review template

## Quick Start

```bash
node scripts/plan.js --morning
node scripts/plan.js --evening
node scripts/plan.js --tasks "Deep work; Standup; Code review; Update docs" --morning
```

## Use when

Starting each day with intention, building a planning habit, structuring work around energy.
